# 🎵 Sound Pad Pro

Полнофункциональное приложение для создания и управления звуковыми подушками (Sound Pad) с интерактивными кнопками.

## ✨ Возможности

### 🎮 Основной функционал
- ✅ **Play/Pause** - интерактивные кнопки воспроизведения и паузы для каждого звука
- ✅ **Загрузка файлов** - поддержка MP3, WAV, OGG и других аудиоформатов
- ✅ **Динамическая сетка** - автоматическое расположение кнопок
- ✅ **Остановка всех звуков** - кнопка для экстренной остановки

### ⚙️ Режим редактирования
- ✅ **Изменение названия** - переименование звуков
- ✅ **Выбор цвета** - 8 предустановок + свободный выбор цвета
- ✅ **Зацикливание** - опция для повторного воспроизведения
- ✅ **Регулировка громкости** - 0-100%
- ✅ **Скорость воспроизведения** - 0.5x - 2x
- ✅ **Горячие клавиши** - назначение клавиш 1-9, Q-O для быстрого доступа
- ✅ **Задержка перед воспроизведением** - добавление паузы в секундах
- ✅ **Удаление звуков** - удаление отдельных файлов
- ✅ **Предпросмотр** - встроенный плеер для проверки перед сохранением

### 💾 Управление проектами
- ✅ **Сохранение проекта** - экспорт всех параметров в `.soundpad` файл
- ✅ **Загрузка проекта** - импорт сохраненных параметров
- ✅ **Автосохранение** - сохранение в localStorage браузера
- ✅ **Очистка всех звуков** - полный сброс проекта

### 🎨 Интерфейс
- ✅ **Красивый дизайн** - современный градиентный интерфейс
- ✅ **Адаптивный дизайн** - работает на ПК, планшетах и мобильных устройствах
- ✅ **Интуитивное управление** - легко найти все функции
- ✅ **Модальное окно редактирования** - удобное редактирование параметров

## 🚀 Быстрый старт

1. **Откройте приложение** - откройте `index.html` в браузере
2. **Загрузите звуки** - нажмите на поле выбора файлов и выберите аудиофайлы
3. **Воспроизведите** - нажимайте на кнопки для проигрывания звуков или используйте клавиши 1-9, Q-O
4. **Редактируйте** - нажмите "Режим редактирования" для изменения параметров
5. **Сохраните** - используйте кнопку "Сохранить проект" для экспорта

## ⌨️ Горячие клавиши

### По умолчанию:
- **1-9** - воспроизведение звуков 1-9
- **Q-O** - воспроизведение звуков 10-18

### В режиме редактирования:
- **✏️** - редактировать параметры звука
- **🗑** - удалить звук

## 🎯 Параметры звука

Каждый звук имеет следующие настраиваемые параметры:

| Параметр | Диапазон | Описание |
|----------|----------|----------|
| Название | Текст | Название звука |
| Цвет | 8 + выбор | Цвет кнопки |
| Зацикливание | В��л/Выкл | Повторять ли звук |
| Громкость | 0-100% | Уровень громкости |
| Скорость | 0.5x-2x | Скорость воспроизведения |
| Горячая клавиша | 1-9, Q-O | Клавиша для быстрого доступа |
| Задержка | 0+ сек | Задержка перед воспроизведением |

## 💾 Сохранение и загрузка

### Автосохранение
Приложение автоматически сохраняет все параметры в localStorage браузера. При перезагрузке страницы все настройки восстанавливаются.

### Экспорт проекта
Нажимайте "Сохранить проект" для скачивания файла с расширением `.soundpad`, содержащего все параметры.

### Импорт проекта
Нажимайте "Загрузить проект" для импорта сохраненных параметров. **Примечание:** аудиофайлы нужно загрузить отдельно.

## 📝 Примечания

1. Все параметры звуков сохраняются локально в браузере
2. Размеры аудиофайлов не ограничены, но зависят от браузера
3. Максимум 18 звуков с горячими клавишами (1-9, Q-O)
4. При загрузке проекта убедитесь, что аудиофайлы загружены

## 🌐 Использование

Приложение полностью функционирует в браузере без необходимости сервера. Все данные хранятся локально на устройстве пользователя.
